| [notes](notes/) | Fly API + Supabase PG — `note.add` / `note.list` proves write + read |

See [podfly doc/supabase.md](https://github.com/127thousand/podfly/blob/main/doc/supabase.md).

## Free-tier note

Supabase free plans allow a limited number of **active free projects**. If create fails with a project limit error, pause or delete an unused project (or upgrade), then re-run deploy.
